## 1. 产品简介

**公园四驱 · 圈速打榜** 是一款面向公园/户外迷你四驱车爱好者的微信小程序，主要提供：

- **圈速认定与打榜**：在指定赛道上传成绩视频，经主理人认证后进入公开圈速榜
- **赛道信息**：查看赛道位置、平面图、认定规则，一键导航至现场
- **车友社区**：发帖、评论、点赞、关注，交流改装心得与赛道见闻

### 适用角色

| 角色 | 说明 |
|------|------|
| **车手** | 选择赛道、上传圈速、查看榜单、参与社区互动 |
| **赛道主理人** | 经认证后可创建/管理赛道，审核车手提交的成绩 |
| **游客** | 可浏览赛道、榜单与社区内容；上传成绩、发帖等操作需先登录 |

同一微信账号可以同时是车手和赛道主理人。

## 2. 界面导航

### 2.1 底部 Tab 栏

登录后，小程序底部始终显示四个 Tab：

| Tab | 功能 |
|-----|------|
| **首页** | 身份快捷入口、最近访问赛道 |
| **圈速榜** | 按赛道查看排名 |
| **社区** | 浏览与发布帖子 |
| **我的** | 个人资料与功能入口 |

### 2.2 首页快捷入口

首页提供两个主要入口：

- **我是主理人**：申请认证、创建与管理赛道
- **我是车手**：进入赛道列表，选择赛道挑战圈速

若您曾访问过赛道，首页还会显示「最近赛道」快捷卡片，点击可直接进入对应赛道详情。

> 提示：您也可以不经过首页身份按钮，直接通过底部 Tab 使用圈速榜、社区等功能。

## 3. 账号与登录

### 3.1 登录方式

1. 进入 **我的** Tab
2. 点击 **微信登录**
3. 授权后完成登录

部分功能（上传成绩、发帖、关注等）需要登录后才能使用。未登录时点击相关操作，系统会提示您先登录。

### 3.2 修改个人资料

路径：**我的** → 点击头像区域 → **修改资料**

可修改内容：

| 项目 | 说明 |
|------|------|
| 头像 | 点击头像，使用微信头像选择器更换 |
| 昵称 | 最多 32 字 |
| 个人简介 | 最多 200 字，在社区用户详情页展示 |

### 3.3 退出登录

路径：**我的** → **退出登录**

## 4. 车手功能

### 4.1 浏览与选择赛道

**入口：**

- 首页 → **我是车手**
- **圈速榜** Tab → 选择赛道

**赛道列表** 展示各赛道卡片，包含赛道名称、距离（需授权位置权限）、榜单 Top1 圈速、入榜人数等信息。

支持 **搜索赛道名称** 快速查找。

### 4.2 赛道详情

点击赛道卡片进入详情页，可查看：

| 信息 | 说明 |
|------|------|
| 赛道名称、主理人、长度 | 点击标题区域可展开/收起更多详情 |
| 赛道平面图 | 支持左右滑动浏览，点击可放大预览 |
| 示例认证视频 | 主理人提供的拍摄参考 |
| 认定说明 | 文字规则补充 |
| 赛道位置 | 地图展示，点击地址可打开系统地图导航 |
| 联系主理人 | 若有联系方式，点击可复制 |

**底部操作：**

- **上传我的成绩**：提交圈速认定材料
- **查看圈速榜**：查看该赛道排名

> 上传前请先观看主理人的示例视频与认定说明，按规则拍摄 **完整一圈** 的视频。

### 4.3 上传圈速成绩

**前置条件：** 已登录，且已进入某一赛道详情页或圈速榜页。

**操作路径：** 赛道详情 → **上传我的成绩**，或圈速榜 → **上传成绩**

#### 必填项

| 字段 | 说明 |
|------|------|
| **圈速** | 格式为 `分:秒.毫秒`，例如 `0:32.580` 表示 32 秒 580 毫秒；不足 1 分钟也可只填 `32.580` |
| **成绩认定视频** | 完整一圈的实拍视频，建议清晰展示起终点 |

#### 选填项

| 字段 | 说明 |
|------|------|
| 车型配置单 | 文字描述或图片（马达、电池、改装要点等） |
| 车辆照片 | 最多 3 张 |
| 备注 | 不超过 100 字 |

填写完成后点击 **确认提交**。

#### 提交后流程

1. 页面提示「上传成功，等待主理人认证」
2. 成绩进入 **审核中** 状态，**暂不上榜**
3. 主理人审核通过后，成绩出现在圈速榜；若被拒绝，可在「我的成绩」中查看原因

您可在提交成功后点击 **查看我的成绩** 跟踪审核进度。

### 4.4 查看圈速榜

**入口：** 底部 **圈速榜** Tab

1. 点击顶部赛道名称，从列表中选择要查看的赛道
2. 榜单按圈速从快到慢排列；圈速相同时，先提交者排名靠前
3. 每条记录显示排名、昵称、圈速，点击进入 **成绩详情**
4. 若您在该赛道有已认证的最佳成绩，底部显示 **我的排名**

**提示：** 若有成绩正在审核中，底部会提示「你有 N 条成绩审核中，通过后显示排名」。

### 4.5 查看成绩详情

点击榜单中的任意条目，或从 **我的成绩** 进入，可查看：

- 车手昵称与头像
- 圈速、提交时间、排名（已认证时）
- 审核状态（审核中 / 已认证 / 未通过）
- 主理人留言（若有）
- 成绩认定视频
- 车型配置单、车辆照片、备注

圈速榜上的成绩均为公开信息，便于社区监督与核实。

### 4.6 我的成绩

路径：**我的** → **我的成绩**

列表展示您提交过的所有成绩，包含：

- 所属赛道
- 审核状态（待审核 / 已通过 / 已拒绝）
- 圈速与提交时间
- 已通过成绩的排名
- 被拒绝时的主理人备注

点击任意条目可查看完整详情。

## 5. 赛道主理人功能

成为赛道主理人需先提交认证申请，审核通过后方可创建赛道。

### 5.1 申请赛道主理人

**入口：**

- 首页 → **我是主理人**
- **我的** → **申请赛道主理人**（尚未认证时显示）

#### 填写信息

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓名 | 是 | 与身份证一致的姓名 |
| 身份证号 | 是 | 18 位身份证号 |
| 手机号 | 是 | 便于线下联系核实 |
| 微信号 | 否 | 便于联系 |

提交后进入 **主理人申请状态** 页，可查看审核进度。

#### 审核说明

- 提交后需等待 **线下核实**，请保持手机畅通
- 状态包括：**审核中**、**已通过**、**未通过**
- 通过后，「我的」页会显示「赛道主理人」标识，并可创建赛道
- 若未通过，可查看审核备注并 **重新申请**

### 5.2 主理人工作台

**入口：**

- 申请通过后，首页 **我是主理人**
- **我的** → **我管理的赛道**
- **我的** → **新建赛道**

工作台展示您创建的所有赛道，每条赛道可：

- 查看 **待审核** 成绩数量，点击进入审核
- **编辑赛道** 信息

### 5.3 创建 / 编辑赛道

路径：**我管理的赛道** → **+ 新建赛道**，或点击 **编辑赛道**

表单分三步完成：

#### 第 1 步：基本信息

| 字段 | 必填 | 说明 |
|------|------|------|
| 赛道名称 | 是 | 建议 ≤ 20 字，如「XX 公园北广场赛道」 |
| 赛道位置 | 是 | 在地图上点击选点，或使用「搜索选点」 |
| 所属主理人 | 是 | 默认填充您的昵称 |
| 主理人联系方式 | 否 | 微信号或手机号，展示在赛道详情页 |

#### 第 2 步：赛道详情

| 字段 | 必填 | 说明 |
|------|------|------|
| 赛道长度 | 否 | 单位：米 |
| 赛道平面图 | 否 | 最多 3 张 |

#### 第 3 步：认定说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 成绩认定示例视频 | 否 | 告诉车手如何拍摄有效成绩，建议 ≤ 60 秒 |
| 文字规则补充 | 否 | 最多 500 字 |

填写完成后点击 **提交**，即可在赛道详情页查看效果。

### 5.4 审核车手成绩

路径：**我管理的赛道** → 某赛道 **待审核 (N)** → 点击待审核条目

#### 审核页面内容

- 车手昵称、提交圈速、提交时间
- 成绩认定视频
- 车型配置单、车辆照片、备注（若有）

#### 审核操作

| 操作 | 说明 |
|------|------|
| **通过** | 确认或修正 **认定圈速**（若视频中的实际圈速与车手填写不一致，可在此更正），可选填审核备注 |
| **拒绝** | 必须填写拒绝原因（最多 200 字） |

审核完成后：

- **通过**：成绩进入圈速榜，车手收到站内信通知
- **拒绝**：成绩不上榜，车手可在「我的成绩」中查看拒绝原因

可在 **成绩审核** 页切换「待审核 / 已通过 / 已拒绝 / 全部」查看历史记录。

## 6. 社区功能

### 6.1 浏览帖子

**入口：** 底部 **社区** Tab

- 顶部切换 **板块**（赛道/赛事专区、新手入门区、车手交流区、新品发布区）

### 6.2 发帖

**入口：** 社区页右下角 **发帖** 按钮（需登录）

| 字段 | 必填 | 说明 |
|------|------|------|

| 功能 | 操作 |

| **评论点赞** | 对单条评论点赞 |

路径：**我的** → **我的关注**

展示您在社区中关注的用户列表。在社区帖子中关注感兴趣的车友，便于后续查看其动态。

## 7. 站内信

路径：**我的** → **站内信**

站内信用于接收系统通知，例如：

- 成绩审核结果（通过 / 拒绝）

功能说明：

## 8. 权限与隐私说明

### 8.1 位置权限

小程序会在以下场景请求位置权限：

- 计算您与赛道的距离
- 在地图上显示当前位置
- 导航至赛道

首次使用时会弹出授权说明，您可以在微信设置中随时管理授权。

### 8.2 内容公开性

- 圈速榜上的成绩视频、配置单、车辆照片均为 **公开内容**
- 主理人联系方式仅在赛道详情页展示，由主理人主动填写
- 个人手机号、身份证号等认证信息 **不会** 公开展示

### 8.3 内容规范

请上传真实、合规的内容。虚假成绩、不当言论可能影响您的使用体验，严重者可能被限制功能。

## 9. 常见问题

### Q1：为什么上传成绩后没有出现在圈速榜？

成绩提交后需经 **赛道主理人审核**。审核通过前状态为「审核中」，不会上榜。请耐心等待，或在「我的成绩」中查看状态。

### Q2：主理人修改了我的圈速，为什么和提交的不一样？

主理人审核时可依据视频认定实际圈速，若与您填写的数值不一致，会以 **认定圈速** 为准上榜。详情页会同时展示提交值与认定值。

### Q3：圈速格式怎么填？

推荐使用 `分:秒.毫秒` 格式，例如：

- `0:32.580` — 32 秒 580 毫秒
- `1:05.120` — 1 分 5 秒 120 毫秒
- `32.580` — 不足 1 分钟时可省略分钟部分

### Q4：如何成为赛道主理人？

在 **我的** → **申请赛道主理人** 填写真实信息并提交。审核通过后即可创建赛道。请确保手机号畅通，以便线下核实。

### Q5：可以同时是车手和主理人吗？

可以。同一微信账号认证为主理人后，仍可像普通车手一样上传成绩、参与社区。

### Q6：没有赛道怎么办？

若所在区域尚无赛道，您可以 **申请成为赛道主理人** 并创建第一条赛道；也可以关注社区「赛道/赛事专区」，了解附近车友的动态。

### Q7：如何联系赛道主理人？

进入 **赛道详情** 页，若主理人填写了联系方式，可在页面底部 **联系主理人** 处点击复制。

## 10. 快速上手路径

### 车手：第一次上传成绩

```
首页 → 我是车手 → 选择赛道 → 赛道详情
→ 查看认定说明与示例视频 → 上传我的成绩
→ 填写圈速 + 上传视频 → 确认提交
→ 等待主理人认证 → 圈速榜查看排名
```

### 主理人：从零到上线赛道

```
我的 → 申请赛道主理人 → 填写信息 → 提交申请
→ 等待认证通过 → 我管理的赛道 → 新建赛道
→ 分步填写赛道信息 → 提交
→ 待审核 → 审核车手提交的成绩
```

### 社区：发帖与交流

```
社区 Tab → 选择板块 → 发帖 → 填写标题与正文 → 发布
→ 在帖子详情点赞、评论、关注作者
```
