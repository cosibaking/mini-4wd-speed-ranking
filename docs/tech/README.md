## 阅读顺序建议

1. **架构设计** — 了解分层、模块边界与依赖方向  
2. **数据库设计** — 理解实体关系  
3. **API 接口总览** — 前后端协作契约  
4. 按需阅读各 **模块文档** 进行实现

## 技术栈摘要

| 层级 | 选型 |
|------|------|
| 客户端 | 微信小程序（原生 WXML + TypeScript） |
| 服务端 | Node.js 20 + Koa 2 + TypeScript |
| 数据库 | MySQL 8.0 |
| 缓存 | Redis 7（会话、榜单缓存、热门帖） |
| 对象存储 | 腾讯云 COS + CDN |
| 鉴权 | 微信 `code2Session` + JWT |
